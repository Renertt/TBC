# Terrain-and-buildings
Unciv mod adding new terrain and building types

## Current Features

### Ancient Era
* **Peat:** A new bonus resource that can be found exclusively on **Marsh** tiles. It provides `+1 Production` to the base tile.
* **Peat Cutter:** A worker improvement available with **Mining**. It can be built on Marsh tiles containing Peat, yielding an additional `+2 Production` and `+1 Gold`.
